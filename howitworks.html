This script consists of an executable python file. In order to download the file you have to fill out the form located in the Generate your day off section. Make sure that you are already logged-in to the g-mail acccount you'll be using to join the meeting with, otherwise, the script will not work. When you click on the download button, a custom script will be downloaded to your computer and the only thing you'll have to do is double-click the file at least 5 minutes before the meeting. The dependencies included in this script are:

  • Selenium
  • SeleniumBase
  • time
  • datetime
